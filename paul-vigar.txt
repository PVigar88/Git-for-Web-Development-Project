Stretch Questions

1. What is the difference between an inline element and a block element?
    Inline elements do not start a new line and have width's that are dependant on internal contents
    Block elements start a new line and have full width available as defined by their parent's width.
2. What happens when an element is positioned absolutely? 
    an absolutely positioned element won't move when the window is resized.

5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
    The Box Model is the term for box that exists around content in html.  Margins, Border, and padding make up the layers that exist around any content displayed 
    on a web page.  box-sizing: border-box is a "solution" that incorporates padding and borders into the width although margins are still separate.
